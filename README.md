# langchain-demo
A small repo for playing with langchain

# Quick Start

## Pre-requisites
- python 3.13
- [Google AI Studio API key](https://aistudio.google.com/app/api-keys)
- Add your `GOOGLE_API_KEY` to a `.env` file at the root of the project
- Install Docker desktop

## Install 
```bash
make install
```

## Run
```bash
poetry run python langchain_demo/quickstart.py
```

```bash
make install
make build
make run
```
```bash
curl 127.0.0.1:8080
```
